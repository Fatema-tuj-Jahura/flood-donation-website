Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface provided by web browsers that allows scripts to access and manipulate the content, structure, and style of HTML and XML documents.

How do you select an element from the DOM?

By using getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows you to handle events at a higher level in the DOM rather than directly on the target elements themselves. It leverages the concept of event bubbling, where an event propagates up from the target element through its ancestors in the DOM tree. It improved performance, handled dynamic content.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, we can use JavaScript methods like setAttribute() to set attributes or getAttribute() to retrieve them. We can also directly change properties, such as element.id or element.className. For styles, we can modify inline styles using element.style or use the classList property to add, remove, or toggle CSS classes. Additionally, we can update content using innerHTML or textContent and dynamically add or remove elements with methods like createElement() and appendChild().
